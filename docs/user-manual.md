# 用户手册

## 客户端使用

### 启动与登录
1. 双击桌面快捷方式启动
2. 输入启动密码（首次使用需要设置密码）
3. 支持虚拟键盘输入（防止键盘记录）
4. License验证通过后进入主界面

### 仪表盘
主界面分为四个状态卡片：
- **硬件状态**: 显示硬件检测结果，绿色=正常，黄色=异常
- **事件日志**: 显示的日志总量
- **风险项**: 反作弊扫描发现的可疑项目数
- **报告状态**: 当前报告生成和上传状态

快捷操作区提供一键扫描和快速硬件检测。

### 硬件信息
- 分卡片展示CPU、GPU、主板、硬盘、网卡、内存详情
- 硬件属性标注是否被伪造/篡改
- "重新采集"按钮刷新硬件数据
- 硬件变动历史追踪

### 行为时间线
- 以时间倒序展示所有用户操作
- 事件类型包括: 进程运行、文件操作、USB插拔、软件安装、系统变更
- 支持按类型/等级筛选
- 上方柱状图展示事件分布趋势

### 反作弊扫描
- 点击"启动深度扫描"进行全面检查
- 扫描包括: 进程扫描、服务/驱动扫描、内核层检测、注册表分析
- 扫描结果按严重度分类（严重/高危/可疑/注意）
- 每个风险项显示匹配规则、发现时间和状态

### 报告管理
- "生成新报告"聚合所有采集数据
- 支持预览、导出PDF/HTML
- 报告自动AES加密
- "上传审核"将报告加密上传到服务器

## 审核面板使用

### 登录
访问审核面板URL，使用审核员账号登录

### 案件管理
- 列表展示所有已上传案件
- 支持按状态、优先级、时间筛选
- 点击案件查看详情

### 报告审核
- 查看解密后的完整报告
- 按时间线浏览所有事件
- 对比多份报告
- 标记嫌疑项（可疑/确认/良性/误报）
- 添加审核备注
