## 数据流

### 采集流程
1. 用户点击按钮触发采集
2. React → IPC → Electron 主进程
3. 主进程 spawn native 采集器 (stdout JSON Lines)
4. 采集器通过 stderr 报告进度
5. 采集完成返回完整 JSON
6. 主进程通过 IPC 返回给渲染进程
7. React 更新 Store → 页面

### 上传流程
1. 报告生成 → 本地 JSON 聚合
2. AES-256 加密报告数据
3. TLS 上传到 FastAPI 服务器
4. 服务器解密、验证、存储到 PostgreSQL
5. 返回报告ID和状态
6. 审核员通过 Web 面板查看

## 安全设计

- **传输安全**: TLS 1.3, 载荷加密(AES-256-GCM), 请求签名(HMAC)
- **存储安全**: 本地 SQLCipher, 服务端 PostgreSQL 加密
- **访问控制**: JWT 认证, 角色权限(admin/auditor/supervisor)
- **代码保护**: Electron asar加密, .NET ConfuserEx, C++ VMProtect
